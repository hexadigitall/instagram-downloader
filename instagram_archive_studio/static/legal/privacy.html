Privacy Notice

Social Archive Studio is designed as a local-first app. Job records, downloaded media, metadata, ZIP archives, and exports are stored on the machine where you run the software.

Data Stored Locally

The app stores job IDs, social media URLs, platform names, media identifiers, profile names, status messages, logs, metadata returned by the downloader, and generated files inside the configured downloads directory.

Session Files

If you provide an Instaloader session file path, the app reads that local file so Instaloader can access owner-authorized archive workflows. Do not use another person's session file or account.

No Hosted Service Yet

This local version does not include cloud accounts, analytics, payment processing, hosted content storage, or team collaboration. Add a production privacy policy before adding any hosted features.

Exports

CSV and JSON exports may include URLs, usernames, captions, local file names, and job logs. Treat exports as sensitive if they include client, creator, or private workflow information.
